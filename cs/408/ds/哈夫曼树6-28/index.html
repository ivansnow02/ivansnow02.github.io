# 题目描述

哈夫曼树，第一行输入一个数 n，表示叶结点的个数。需要用这些叶结点生成哈夫曼树，根据哈夫曼树的概念，这些结点有权值，即 weight，题目需要输出所有叶子结点的路径长度与权值的乘积之和。

# 输入

输入有多组数据。
每组第一行输入一个数 n，接着输入 n 个叶节点（叶节点权值不超过 100，2<=n<=1000）。

# 输出

输出权值。
