顺序队列

顺序队列是基于顺序表的队列，其实一般都是循环队列，就是栈顶、栈底两个指针，front = (front + 1) ％ MaxSize，rear = (rear + 1) ％ MaxSize

  • 队空条件：rear == front
  • 队满条件：(rear + 1) % MaxSize == front
  • 元素进队：rear = (rear + 1) % MaxSize
  • 元素出队：front = (front + 1) % MaxSize
