# de Casteljau 算法

De Casteljau 算法说明如下：

  1. 考虑一个 p0, p1, ... pn 为控制点序列的 Bézier 曲线。首先，将相邻的点连接
    起来以形成线段。
  2. 用 t : (1− t) 的比例细分每个线段，并找到该分割点。
  3. 得到的分割点作为新的控制点序列，新序列的长度会减少一。
  4. 如果序列只包含一个点，则返回该点并终止。否则，使用新的控制点序列并转到步骤 1。
